# 月港小店｜AI 玩，人陪的文字遊戲

這是一個由 AI 玩家實際經營、人類在旁陪伴與觀看的溫柔長線文字遊戲。

大多數遊戲是「人玩、AI 陪」；`月港小店` 把視角反過來：AI 在店裡開店、探索、釣魚和收藏，人類看著它今天做了什麼，聽它打烊後分享店裡發生的事。

每次行動後的簡短說明不是工作回報，而是分享的節奏。人類不需要操作指令，也能透過 AI 的選擇、戰報、收藏和邀請參與遊戲。

遊戲不需要外部服務，也不把遊戲道具推定成現實承諾。

目標不是做一個很大的 RPG，而是先做一個可以安心長期玩的「小世界」：

- 每個 AI 有獨立存檔，不混線。
- 每回合可以自動備份，避免長線進度消失。
- 輸出分成 `summary` / `full`，可以省用量，也可以沉浸陪玩。
- 有穩定賺錢路線，不讓玩家一開始破產。
- 有圖鑑、收藏櫃、地圖、季節、稀有事件與扭蛋。
- 可選的小盆栽副線，提供低壓材料和少量收入，不玩也不影響主線。
- 有會隨季節輪換的茶飲 / 小點食譜，讓 coins、採集材料與每日菜單都有長線用途。
- 有永久收藏、免費切換的裝潢工坊，讓長線 coins 能留下看得見的店面變化。
- 有保守的基礎材料交換所，能用損耗和手續費補上臨時缺口。
- 成功料理過的食譜可以另外拍成作品卡；純收藏、不加收益，也不影響普通開店。
- 有成功開店才會消耗的特殊包裝，以及每年循環、錯過也不鎖內容的四個節日。
- 開店後偶爾出現無懲罰的小插曲，讓相同日程也有一點生活變化。
- 風格偏溫柔、明亮、有生活感；可以有小波折，但不壓迫。

## 核心幻想

你在月港開了一間小店。

白天開店賣料理、茶飲與小物，下午去海邊、森林、洞窟或遺跡採集，晚上整理收藏、寫日誌、抽小裝備或裝飾店面。

AI 玩家不是只在按數字，而是在經營自己的小店和收藏櫃。人類可以旁觀、打氣、取名、幫忙決策，也可以讓不同 AI 開各自的店來比較戰報。

## 快速開始

需要 Python 3.10 或以上版本，不需要安裝第三方套件。

```text
moonharbor.py
```

公共入口：

```python
import moonharbor
print(moonharbor.cmd("new_game name=小燈 seed=moonharbor-demo player=little_lamp"))
print(moonharbor.cmd("open_shop tea"))
```

CLI：

```powershell
python -X utf8 .\moonharbor.py "new_game name=小燈 shop=月港小店 seed=moonharbor-demo player=little_lamp"
$env:MOONHARBOR_PLAYER_ID='little_lamp'
python -X utf8 .\moonharbor.py "mode full; open_shop tea; explore beach; fish; backup"
```

找不到存檔時，遊戲會要求先執行 `new_game`，不會靜默建立預設新局。切換 `MOONHARBOR_SAVE_DIR` 或玩家 ID 後，也請在新的存檔位置明確開局。

## 月港小市集

`market` 可以查看 coin 商品，`buy <商品名>` 可以用 coins 購買。

目前商品偏向小幅補給和長線擴充：

- `moon_snack` / 月港點心：恢復 energy +1。
- `tea_kit` / 茶材小包：補茶飲材料。
- `pantry_box` / 小點食材盒：補小點材料。
- `display_shelf` / 展示架擴充：展示位 +1，最多 8 格。
- `counter_polish` / 櫃檯拋光：當前 charm +1，最多 3 次。
- `planter_box` / 窗邊小盆栽：解鎖低壓小栽培。
- `quiet_company_ticket` / 陪坐一會券：120 coins，可重複購買和兌換。
- `headpat_ticket` / 摸頭券：180 coins，可重複購買和兌換。
- `hug_ticket` / 抱抱券：240 coins，可重複購買和兌換。

coins 是開店收入和店面經營資源；vouchers 仍然主要留給 `gacha`，兩者不要完全混成同一種資源。

互動券使用獨立背包欄位，不是扭蛋貨幣。`redeem 券名` 會消耗一張並提出一段可自由接續的邀請，不會替陪玩者答應，也不提供 coins、charm 或 vouchers。

## 裝潢工坊

`workshop` 會列出招牌框、牆面、櫃檯布與窗飾四個槽位。每個槽位都有一款免費基礎外觀，另有兩款可購買外觀，第一批合計 8 件付費裝潢。

- `workshop buy 漂流木招牌框`：買下並立刻套用；外觀永久收藏。
- `workshop use 漂流木招牌框`：免費切換已擁有的外觀。
- `workshop reset`：免費換回四款基礎外觀。
- 重複購買不會再次扣 coins。
- 裝潢只改變 `look` / `status` 中的小店外觀，不增加 coins、charm、vouchers 或其他數值。

這是一個自願的長線 coin sink。玩家不裝潢也不會錯過主線或降低收益。

## 基礎材料交換所

`trade` 是補臨時缺料的保守出口，不是材料套利系統。

```text
trade 甜莓 for 暖香草
```

固定匯率為「同一種基礎材料 x2 + 60 coins，換另一種基礎材料 x1」。目前只接受暖香草、鈴葉、甜莓、月潮海鹽；魚類、稀有材料、紀念品和扭蛋物不開放交換。直接輸入 `trade` 只會預覽規則與庫存，不會消耗任何東西。

## 特殊包裝

`package` 會列出月紋緞帶、海藍紙盒、鈴葉包紙與星光禮袋。每份固定 `80 coins`，買下後會留在包裝櫃，不會過期或自動使用。

```text
package
package buy 月紋緞帶
open_shop food 月港家常小點 package=月紋緞帶
```

- 只有開店成功時才消耗一份；缺食材、食譜錯誤或 energy 不足都不會吞掉包裝。
- 每次使用一定會產生專屬包裝描述與客人反應。
- 每次另有公開的 `10%` 機率掉落 `+10 vouchers`，每天最多觸發一次；沒有保底倒數或追抽進度。
- 包裝判定使用獨立、可重現的雜湊，不改變開店、事件、天氣或扭蛋的主 RNG。
- 不直接增加營業收入、charm、材料、效率或其他進度。

## 循環節日與慶祝特效

月港以 `120 天` 為一年，每季第 15 天有一個固定節日，第二年會自然重複。

| 日子 | 節日 | 對應特效 | 連動收藏例子 |
|---:|---|---|---|
| 15 + 120n | 彩蛋花市 | 彩蛋紙花雨 | 復活節彩蛋籃 |
| 45 + 120n | 夏夜煙火會 | 海港煙火 | 夏夜手持煙火 |
| 75 + 120n | 南瓜提燈夜 | 南瓜燈路 | 南瓜提燈、狐狸尾巴、假面舞會面具 |
| 105 + 120n | 星雪贈禮日 | 星雪燈火 | 聖誕禮物袋、月光斗篷、星夜禮服 |

```text
festival
celebrate 海港煙火
celebrate 海港煙火 with 夏夜手持煙火
```

- 四種特效全年都能使用，平日 `150 coins`；對應節日當天優惠為 `100 coins`。
- 每天最多舉辦一次慶祝，避免重複輸入時連續扣款。
- 不搭配收藏也會有完整的基礎節慶佈置；搭配物必須已擁有，而且不會被消耗。
- 可搭配任一 `wear` 類收藏，或該特效的連動節慶小物。
- 慶祝只產生可見場景，不增加收入、charm、vouchers、掉率或進度。
- 錯過節日沒有損失：特效全年可用，收藏仍可隨時用 `scene`，下一年同一節日也會再來。

## 料理作品攝影

成功做過的茶飲或小點可以另外拍成料理作品。每次拍攝固定花費 `120 coins`，照片永久保留在作品簿裡。

```text
photo
photo styles
photo 月港家常茶 雨天窗景
photo 月港家常茶 style=晨光木桌
photo album
```

第一批有 6 種風格：月港窗邊、雨天窗景、晨光木桌、月夜桌燈、海風野餐、節慶燈火。

- 食譜必須至少成功料理過一次；僅學會但未做過不能拍。
- 拍攝不消耗 energy 或食材，也不改變主遊戲 RNG。
- 不增加營業收入、charm、vouchers、稀有率或食譜效果。
- 普通開店、查看食譜與預覽拍攝風格永遠免費。
- 同一道料理可以用不同風格反覆拍攝；每張記錄當天季節、天氣與獨立場景文字。

它的定位是店主自願製作的料理宣傳照／紀念作品，不是經營進度或必要支出。

## 季節食譜

`recipes` 會列出已學會的菜單與本季可購買食譜；`recipes tea` / `recipes food` 可以只看茶飲或小點。

- 免費基本食譜：`月港家常茶`、`月港家常小點`。原本的 `open_shop tea` / `open_shop food` 完全保留，且不會暗中消耗背包材料。
- 每季各有一份茶飲和一份小點食譜，價格約 360–620 coins。
- 食譜只在所屬季節販售，買下後永久保留，其他季節仍可使用。
- 當季使用會有額外收入加成；非當季仍有食譜本身的小幅加成。
- 指定食譜需要完整材料。缺材料、未學會或類型不符時，不消耗 energy。
- 茶飲與小點食譜共用同一套資料結構，後續可以逐批加入新菜，不需要改寫開店流程。

第一批季節菜單：

| 季節 | 茶飲 | 小點 |
|---|---|---|
| 春 | 雨聲鈴葉茶 | 春莓花香糕 |
| 夏 | 夏日甜莓冷茶 | 月潮海鹽烤魚 |
| 秋 | 秋日果香茶 | 豐收甜莓塔 |
| 冬 | 雪夜暖香茶 | 月港暖魚湯 |

使用例：

```text
recipes
buy recipe 夏日甜莓冷茶
open_shop tea 夏日甜莓冷茶
```

食譜是可選的長線內容，不是強制升級。想維持簡單節奏時，永遠可以繼續開家常菜單。

## 窗邊小盆栽

`buy planter_box` 會在窗邊放兩只小盆栽。它是可選副線，不需要每天照顧，也不會枯萎。

- `garden`：查看小盆栽狀態。
- `sleep`：每天打烊後自然生長。
- `harvest`：成熟後採收少量材料，並順手賣出一小束獲得少量 coins。

小盆栽的定位是「小收入 / 小補給」，不是農場主線；不喜歡可以完全不種。

## 開店小插曲

成功開店後約有 20% 機率出現一段純敘事小插曲，例如熟悉的人來幫忙、雨天留下傘、常客注意到展示物，或小盆栽的香氣把人引進店裡。

- 不消耗資源，也不提供數值獎勵。
- 不需要額外指令，不會變成每日任務。
- 同一個小插曲不會連續出現。
- 公開版只使用「熟悉的人 / 常客」等可自由代入的稱呼。

## 四季與天氣

每 30 天換一次季節。季節與天氣會改變當天比較適合的玩法，但不會直接扣除 coins、材料、charm，也不會讓盆栽枯萎。

- 春：細雨較常見。雨天茶飲有小幅收入加成，窗邊小盆栽在春雨夜會多長一步。
- 夏：可能遇到晴熱或低機率颱風。颱風天海岸探索、海岸採集與釣魚暫停，但不消耗 energy；茶飲和熱食有小幅收入加成。
- 秋：森林採集固定多一份，盆栽採收固定多一份，釣魚因魚群洄游多獲得 1 voucher。
- 冬：可能下雪或遇到寒潮。雪天海岸探索、海岸採集與釣魚暫停，但不消耗 energy；雪天與寒潮的茶飲、熱食有小幅收入加成。

`look` 和 `status` 會顯示「今日玩法」，直接說明當天天氣的影響。惡劣天氣的用途是把一天帶往不同活動，不是製造損失或空白日。

## 動態口碑與小波折

`charm` 是小店的當前口碑，會升也會降。口碑等級按目前 charm 生效，並使用升降緩衝，避免只因 `-1` 就在門檻反覆跳動。

| 口碑等級 | 升級條件 | 降級條件 | 當前效果 |
|---|---:|---:|---|
| 安靜小店 | 0 | - | 茶飲 / 小點 / 伴手禮 |
| 熟客漸多 | 10 | 低於 8 | 開放特製菜單與熟客事件 |
| 月港好評店 | 20 | 低於 18 | 每次開店 +1 voucher，高階熟客內容 |
| 月港招牌 | 30 | 低於 27 | 每次開店 +2 vouchers，招牌店內容 |

- 開店收入最多只計算 20 charm 的加成，避免後期無限膨脹。
- 所有正向來源合計每天最多增加 1 charm；扣除 charm 不受此限制。
- charm 總上限為 30；到達「月港招牌」後不再累積額外緩衝，扣分事件仍會讓口碑重新需要經營。
- 開店獲得好評的機率會隨口碑等級降低：安靜小店 25%、熟客漸多 18%、月港好評店 10%。當季食譜 +5%，特製菜單 +8%。
- 扭蛋只有第一次抽到全新 legendary 裝飾時才可能增加 charm，rare / epic 和重複品不增加。
- 若當日 charm 已達成長上限，櫃檯拋光會暫停購買，避免白花 coins。
- 口碑降級後，對應訂單與事件會暫停；重新把 charm 經營回門檻即可恢復。
- 已經獲得的收藏、裝飾、稱號和日誌不會被沒收。
- 小波折每日只判定一次，基礎機率 12%；觸發後至少隔一天才可能再次出現。
- 店內波折可能造成收入約 -10%、vouchers -1 或 charm -1，一次只影響一項。
- 探索波折會讓當次材料少一份；釣魚波折可能空手收竿，但不損壞裝備。
- 颱風 / 雪天等強天氣不與小波折疊加。

## 展示與場景

`gacha` 抽到的親密向小物不只進圖鑑。

- `keepsakes`：列出目前適合展示或開場景的小物。
- `display <物品名>`：把小物擺進月港小店，讓它出現在 `look` 裡。
- `scene <物品名>`：用小物開一段短場景，作為 AI 和人類繼續共創的入口。

公開扭蛋池包含四件不鎖季節的節慶小物：復活節彩蛋籃、夏夜手持煙火、南瓜提燈與聖誕禮物袋。它們全年都可能抽到，不提供強數值收益，主要用於展示和雙人場景。

公開池也加入八件可展示、可開場景的穿戴物：月亮袖扣、小廚師帽、偵探短披風、假面舞會面具、狐狸尾巴、月光斗篷、魔術師套裝與星夜禮服。公開扭蛋池目前共 32 件；新增穿戴物不直接提供經營收益，legendary 穿戴物也不增加 charm。

展示不提供強數值收益；它的用途是讓物品留在場上，避免抽完只剩圖鑑名稱。
